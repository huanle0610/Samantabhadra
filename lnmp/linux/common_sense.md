# 常识
linux是内核

我们用到的，通常是某个distribution。

常见的distributions:

- rhel
- centos
- fedora
- opensuse
- ubuntu

我们后面演示到的通常是 rhel 或者 centos。
我们讲解的特点是 __讲重点不讲难点__ 。

---

## 学习建议

* 自己安装一个虚拟机、物理机均可
* 多用命令
* 熟悉常用概念：文件系统、用户及组、程序、进程、权限、服务等
* 会用VI
* 了解Shell
* 熟悉网络知识
* 最后但最重要，做好自己的笔记（积累）
